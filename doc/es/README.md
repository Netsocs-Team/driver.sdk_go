# Documentación (ES) — Netsocs Driver SDK (Go)

Esta carpeta contiene documentación **en español** enfocada a:

- **Usuarios / Integradores**: cómo instalar, configurar y operar drivers en Netsocs.
- **Desarrolladores**: cómo crear nuevos drivers con una plantilla estándar, buenas prácticas y checklist de release.

## Índice

- [`USO_SDK_Y_DRIVERS.md`](./USO_SDK_Y_DRIVERS.md): guía para usuarios (instalación/ejecución/configuración/operación).
- [`GUIA_DESARROLLO_DRIVER.md`](./GUIA_DESARROLLO_DRIVER.md): guía exhaustiva para desarrolladores (arquitectura, handlers, objetos, eventos, estados, testing, packaging).
- [`PLANTILLA_DRIVER.md`](./PLANTILLA_DRIVER.md): plantilla y estructura recomendada (qué archivos crear y qué debe ir en cada uno).
